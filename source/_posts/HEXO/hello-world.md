## 快速开始

### 创建一篇新的博客

``` bash
$ hexo new "My New Post"
```

详情: [Writing](https://hexo.io/docs/writing.html)

<!-- more -->

### 启动服务器

``` bash
$ hexo server
```

详情: [Server](https://hexo.io/docs/server.html)

### 生成博客静态文件

``` bash
$ hexo generate
```

详情: [Generating](https://hexo.io/docs/generating.html)

### 发布到服务器

``` bash
$ hexo deploy
```

详情: [Deployment](https://hexo.io/docs/deployment.html)
